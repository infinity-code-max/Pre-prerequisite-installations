# 🚀 Pre-prerequisite Installations

This repository provides automated scripts to install essential prerequisites for DevOps environments across multiple Linux distributions.

---

## 📌 Overview

The scripts in this repository help you quickly set up:

* Docker installations
* change file permission
* Os updates
* kubernetes installations

Supported platforms:

* CentOS
* Ubuntu
* Amazon Linux
* RHEL

---

## ⚙️ Prerequisites

Before running the scripts, ensure:

* You have **sudo/root access**
* Internet connectivity is available
* Git is installed

---

## 📥 Clone the Repository

```bash
git clone https://github.com/infinity-code-max/Pre-prerequisite-installations.git
cd Pre-prerequisite-installations
```

---

## ▶️ Usage

### Step 1: Make script executable

```bash
chmod +x install.sh
```

### Step 2: Run the script

```bash
./install.sh
```

OR

```bash
bash install.sh
```
# Run bash file 

```bash
./filename.sh
```
